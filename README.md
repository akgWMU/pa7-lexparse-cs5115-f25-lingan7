# LexicalAnalyzer_Parser

A lexical analyzer and parser implementation for the SomeWMULife programming language, a simple Pascal-like language.

## Features

- Lexical analysis
- Syntax parsing
- Semantic analysis
- Interpreter
- Test suite

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Run tests:
   ```
   python -m pytest tests/
   ```

## Running Programs

```bash
python -m src.interpreter examples/example.wm
```
